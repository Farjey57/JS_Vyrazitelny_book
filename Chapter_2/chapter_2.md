# Структура программы

## Привязка (переменные)

`let caught = 5 * 5`

Caught - имя привязки. оно используется для захвата значения 5 * 5.

const - постоянная привязка

Именем привязки могут быть любое слово. Начинается не с цифры, может содержать $ _/
